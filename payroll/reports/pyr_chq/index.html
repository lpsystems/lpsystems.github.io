Introduction

This program allows the user to produce a list of cheque listing reports which have been created during previous payroll runs, depending on the scheme code entered.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Cheques Listing Screen

Run A Report

Click ‘Proceed’.

SCHEME CODE - Enter the scheme code required. To get a list of valid scheme codes click the lookup icon. When a valid scheme code has been entered the scheme description will be displayed.

Click ‘OK’ to proceed with the Cheques listing. You can click ‘Cancel’ to return to the proceed menu and abort report selection.

If there are no cheque reports held for the scheme then an error message is displayed informing the user of this. The program then returns back to the main menu.

If there are cheque reports held for the scheme then the following screen is displayed:

Cheques Listing Reports

Using the arrow keys or the scrollbar, select one of the cheque reports listed in the table. Click ‘OK’ to process the report, or click ‘Cancel’ to return to the main menu.

When the selection is complete a message is displayed at the bottom of the screen:

Processing report … Please Wait

When the report has been processed and no data exists in the cheque report an error message is displayed:

The report does not contain any data

The program returns to the main menu.

If data is found in the cheque report then the report continues and is displayed to the screen.

The options for printing are described in the document Printing From Payroll Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
