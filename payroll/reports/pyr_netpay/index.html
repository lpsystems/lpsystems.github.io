Introduction

This program allows the user to produce a list of members and their pay details, depending on the scheme code and dates entered.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Members Pay Report

Run A Report

Click ‘Proceed’.

SCHEME - Enter the scheme code required. To get a list of valid scheme codes click on the lookup icon. When a valid scheme code has been entered the scheme description will be displayed. A scheme code must be entered.

COMPANY - Enter the company code. To get a list of valid company codes click on the lookup icon. When a valid company code has been entered the company description will be displayed.

CATEGORY - Enter the category code. To get a list of valid category codes click on the lookup icon. When a valid category code has been entered the category description will be displayed.

STATUS - Enter the status code. To get a list of valid status codes click on the lookup icon. When a valid status code has been entered the status description will be displayed. The status code must be of a Pensioner type.

SPECIAL CODE - Enter the special code. To get a list of valid special codes click on the lookup icon. When a valid special code has been entered the special code description will be displayed.

When the selection criteria has been entered, the following screen will be displayed. Click ‘Cancel’ to return the user back to the main menu.

Report Dates Screen

DATE FROM - Enter the date you wish to list member’s pay details from. The date must be entered in the format dd/mm/yyyy or ddmmyyyy, and must be within the current tax year.

DATE TO - Enter the date you wish to list member’s pay details to. The date must be entered in the format dd/mm/yyyy or ddmmyyyy, and must be within the current tax year.

Click ‘OK’ to proceed. You can click ‘Cancel’ to return the user back to the main menu.

The following menu options will now be displayed:

  • Summary - This will print a report of total amounts for the stated period.
  • Detailed - This will print a report, listing all amounts within the stated period.
  • Exit - This will return the user back to the main menu.

If Summary or Detailed has been selected then the report starts processing.

When the selection is complete a message is displayed at the bottom of the screen:

Processing report … Please Wait

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

Click ‘OK’ to return to the report menu.

If records are found to match the selection criteria the report continues and is displayed to the screen.

The options for printing are described in the document Printing From Payroll Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
