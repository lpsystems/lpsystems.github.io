Introduction

This program allows the user to restore payroll data, from backed up files, for a specified scheme.

Only users set up as System Administrator can access this program.

If another user is running either the Backup Data program, the Payroll Run program, or the Year End program, then you are unable to process the Year End until the other program has completed.

The following options and screen appear when this program is selected:

  • Restore
  • Exit

Restore Data Screen

Restore Data

Click ‘Restore’.

SCHEME CODE - Enter the scheme code required. To get a list of valid scheme codes click the lookup icon. When a valid scheme code has been entered the scheme description will be displayed. This field must be entered.

Click ‘OK’ to proceed with the backup procedure. You can click ‘Cancel’ to return to the backup menu.

If there are no files containing data for the scheme, then an error message appears informing you of this. You are then returned to the restore menu.

If there are backup files held for the scheme, the table will display a list of dates, times and descriptions.

Using the arrow keys or scrollbar, select one of the files. Click ‘OK’ to continue to restore data, or click ‘Cancel’ to return to the restore menu.

The following message is now displayed to the screen:

You have selected the following data to restore:

Backup Date : dd/mm/yyyy

Backup Time : mm:hh

Run Date : dd/mm/yyyy

Do you wish to continue?

Click ‘Yes’ to carry out the restoration procedure, or ‘No’ to return to the restore menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
