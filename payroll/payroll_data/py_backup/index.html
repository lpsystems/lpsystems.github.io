Introduction

This program allows the user to backup up payroll data for a specified scheme.

Only users set up as System Administrator can access this program.

If another user is running either the Restore Data program, the Payroll Run program, or the Year End program, then you are unable to process the Year End until the other program has completed.

The following options and screen appear when this program is selected:

  • Backup
  • Exit

Selection Screen

Run A Backup

Click ‘Backup’.

SCHEME CODE - Enter the scheme code required. To get a list of valid scheme codes click the lookup icon. When a valid scheme code has been entered the scheme description will be displayed. This field must be entered.

RUN DATE - Enter the run date of the backup in the format dd/mm/yyyy or ddmmyyyy. Click the calendar icon to display a calendar on the screen. This field must be entered. If it is left blank then it will default to today’s date.

BACKUP DESCRIPTION - Enter a description for the backup. This field is not mandatory, however, Payroll Master recommends that a description is entered. This is so that the user will have some idea of what data is contained in the backup file, if they go on to use the Restore Data program at a later date.

Click ‘OK’ to proceed with the backup procedure. You can click on ‘Cancel’ to return to the backup menu.

The following message is now displayed to the screen:

All Payroll data will now be backed up. This may take some time. Do you wish to continue?

If ‘Yes’ is selected, then the backup procedure is carried out. If ‘No’ is selected, then the program returns to the backup menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
