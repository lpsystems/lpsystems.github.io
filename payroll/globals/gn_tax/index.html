Introduction

This program allows the user to globally alter tax codes for members of a specified scheme. It provides a quick and easy way for the user to either increase or decrease the value of a tax code, or replace one tax code with another.

The following options and screen appear when this program is selected.

  • Proceed
  • Exit

Tax Parameters Screen

Run A Tax Code Alteration

Click ‘Proceed’.

Enter the fields as required as follows:

SCHEME - Enter the scheme code you wish to alter tax codes for. This field must be entered.

ALTERATION TYPE - Select the alteration type you require.

Click ‘OK’ to continue with the tax alteration procedure, or click ‘Cancel’ to return to the main menu.

Direct Replacement of one Tax Code with another

When this alteration type is selected, the following screen appears:

Tax Parameters Screen

Enter the fields as follows:

ENTER THE EXISTING TAX CODE - Enter the tax code you wish to replace. This field must be entered.

ENTER THE REPLACEMENT TAX CODE - Enter the new tax code. This field must be entered.

Click ‘OK’ to continue with the tax code alteration, or click ‘Cancel’ to return back to the main menu.

The following screen will now appear:

Tax Parameters Screen

If there are members in the specified scheme who use the existing tax code, then these members will be listed on the screen.

Enter the field as follows:

CHANGE? - All members listed in the table will have this field defaulted to Yes. If you do not wish any of these members to have their tax code changed, then set the field to No.

Click ‘OK’ to accept the tax code alterations, or click ‘Cancel’ to return to the main menu.

Alteration Type

All Tax Codes with a certain Prefix by a fixed number

When this alteration type is selected, the following window is displayed to the screen:

Tax Parameters Screen

Enter the fields as follows:

DO YOU WISH TO INCREASE OR DECREASE THE TAX CODE? - Select to increase or decrease the tax code value.

Click ‘OK’ to continue with tax alteration procedure, or click ‘Cancel’ to return to the main menu.

The following screen will now appear:

Tax Parameters Screen

Enter the fields as follows:

INCREASE/DECREASE THE SCOTTISH TAX CODE - If the tax code you are altering is a Scottish tax code, then enter Yes in this field. If the tax code is not Scottish enter No.

ENTER THE PREFIX OF THE TAX CODES YOU WANT TO INCREASE/DECREASE - Enter a valid Prefix for the tax code you wish to alter. This field must be entered.

ENTER THE AMOUNT YOU WANT TO INCREASE/DECREASE THE TAX CODES BY - Enter an amount that you want to increase/decrease the tax code value by. This field must be entered.

Click ‘OK’ to continue with the tax code alteration, or click ‘Cancel’ to return to the main menu.

The following screen will now appear:

Tax Parameters Screen

If the amount to increase or decrease the tax code value causes the tax code to be an invalid tax code e.g. a negative amount, or too large an amount, the member will not be displayed.

If there are members in the scheme who use a tax code with the stated prefix, then these members will be listed on the screen.

Enter the field as follows:

CHANGE? - All members listed on the screen will have this field defaulted to Yes. If you do not wish any of these members to have their tax code value changed, then un** -tick the field for No.

Click ‘OK’ to accept the tax code alterations, or click ‘Cancel’ to return to the main menu, and no alterations will be processed.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
