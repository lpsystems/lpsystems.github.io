Introduction

This program allows the user to run a year end for a specified scheme. From this program the user can print the P11, P14 and P35 forms.

Only users set up as System Administrator can access this program.

If another user is running either the Backup Data program, the Restore Data program, or the Payroll Run program, then you are unable to process the Year End until the other program has completed.

The following options and screen appear when this program is selected:

  • Run
  • Exit

Tax Year End Screen

Run A Year End

Click ‘Run’.

SCHEME CODE - Enter the scheme code required. To get a list of valid scheme codes click the lookup icon. When a valid scheme code has been entered the scheme description will be displayed. This field must be entered.

ALL EMPLOYEE PAYMENTS/DEDUCTIONS HAVE BEEN PROCESSED - If the payroll run has been processed for the members then tick the box. This field must have been ticked before you can continue onto the next screen.

ALL OF YOUR PAYROLL RECORDS HAVE BEEN UPDATED - If the payroll run data has been committed for the members then tick the box. This field must have been ticked before you can continue onto the next screen.

ALL OF YOUR PAYROLL REPORT(S) HAVE BEEN PRINTED - If the payroll run payslips and payment summary report have been processed for the members then tick the box. This field must have been ticked before you can continue onto the next screen.

YOU HAVE MADE AT LEAST ONE BACKUP COPY OF YOUR DATA - If at least one backup has been taken of the payroll data before you started this program then tick the box. This field must have been ticked before you can continue onto the next screen.

Click ‘OK’ to continue with the year end process, or click ‘Cancel’ to return to the tax year end menu.

The following pop-up window appears:

Proceed with Year End Window

Click ‘Yes’ to continue the year end process or click ‘No’ to return to the tax year end menu.

Printing Year End Forms

The following screen is displayed:

Tax Year End Screen

Enter the fields as follows:

P11 DEDUCTION WORKING SHEET (PAYE) - Click ‘Print’ to display the Pll (PAYE) form to the screen. Further details of the printing options can be found here . When you return to the screen there will be a tick displayed in the box next to the ‘Print’ button.

P11 DEDUCTION WORKING SHEET (NIC) - Click ‘Print’ to display the Pll (NIC) form to the screen. Further details of the printing options can be found here . When you return to the screen there will be a tick displayed in the box next to the ‘Print’ button.

LASER P14 (SHEET 1) - INLAND REVENUE COPY - Click ‘Print’ to display the P14 (SHEET 1) form to the screen. Further details of the printing options can be found here . Please ensure that there are pre-printed P14 Inland Revenue Copy forms in the printer before you send the report to print. When you return to the screen there will be a tick displayed in the box next to the ‘Print’ button.

LASER P14 (SHEET 2) - DSS COPY - Click ‘Print’ to display the P14 (SHEET 2) form to the screen. Further details of the printing options can be found here . Please ensure that there are pre-printed P14 DSS Copy forms in the printer before you send the report to print. When you return to the screen there will be a tick displayed in the box next to the ‘Print’ button.

LASER P14 (SHEET 3) - EMPLOYEE COPY - Click ‘Print’ to display the P14 (SHEET 2) form to the screen. Further details of the printing options can be found here . Please ensure that there are pre-printed P14 Employee Copy forms in the printer before you send the report to print. When you return to the screen there will be a tick displayed in the box next to the ‘Print’ button.

P35 (CS) (SUBSTITUE) - Click ‘Print’ to display the P35 (CS) form to the screen. Further details of the printing options can be found here. When you return to the screen there will be a tick displayed in the box next to the ‘Print’ button.

Export Data to CSV Files - Click ‘Export’ to export payroll data out to CSV files, which can then be used for online submissions. Once this has been done a tick will be displayed in the box next to the ‘Export’ button, and a message will appear to the screen as follows:

Export Data to CSV Files Window

P35 Totals and Declarations

A window will appear called ‘P45 Totals for Tax Year Ending 5th April 2009’. Check that the totals on this screen are correct, and click ‘OK’.

P45 Totals for Tax Year Ending 5th April 2009

Another window will appear called ‘P35 Declarations for Tax Year Ending 5th April 2009 (part one)’. Answer the questions via the radio buttons and click ‘OK’.

P35 Declarations for Tax Year Ending 5th April 2009 (part one)

Another window will appear called ‘P35 Declarations for Tax Year Ending 5th April 2009 (part two)’. Again, answer the questions via the radio buttons and check boxes (if needed) and click ‘OK’.

P35 Declarations for Tax Year Ending 5th April 2009 (part two)

Finally, another window will appear called ‘P35 Declarations for Tax Year Ending 5th April 2009 (part three)’. Answer the questions via the radio button, and choose the capacity in which you sign the declarations. The default is ‘Employer’. Click ‘OK’.

P35 Declarations for Tax Year Ending 5th April 2009 (part three)

Once the user has printed out neccessary reports, and has exported the payroll data, if they are doing online submissions, then they can continue with the year end process.

Click ‘OK’ to continue, or click ‘Cancel’ to return to the tax year end menu.

The following pop-up window appears:

Year End Window

Only select ‘Yes’ if you are happy with all year end report figures. By choosing ‘Yes’, all payroll data of the tax year will be cleared out. If in doubt select ‘No’, which will return you to the tax year end menu.

Backup of Payroll Data

The Year End program provides the user with the option of backing up the payroll data whilst still in the Year End program. The following window is displayed:

Backup Window

Click ‘Yes’ to continue to backup the year end data, or click ‘No’ to return to the tax year end menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
