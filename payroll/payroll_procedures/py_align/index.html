Introduction

Before the first Payroll Run Submission of the new tax year the member’s information needs to be sent to HMRC. This submission registers and validates the member’s information and must be completed for each payroll scheme.

Only users set up as System Administrator can access this program.

The following options and screen appear when this program is selected:

  • Submit
  • Check
  • Exit

Submit

Selecting the ‘Submit’ menu option will allow you to enter the Scheme Code and Sender. You will need the Accounts Office Reference number, Gateway ID and Password that has been provided by HMRC. On clicking ‘OK’ the program will generate an XML file which will then be sent to HMRC and you will receive a confirmation message. This only indicates that HMRC have received the submission and not that the data is valid and has been accepted.

Check

To check the status of a recent submission select the ‘Check’ menu option and enter the Scheme Code. This will ask HMRC the status of the previous submission for that scheme. You will have to wait 10 seconds, a time limit imposed by HMRC, before checking your original submission. The response will either be:

  • A list of errors in a Report Viewer window. Any errors will need to be rectified and the scheme data submitted and checked again.
  • Confirmation that the data has been accepted.
  • A message saying ‘The submission has been acknowledged by the Government Gateway’ indicates that HMRC have received your ‘Check’ request and are currently processing the data submission. You will need to check later to ensure that you receive one of the two responses above.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
