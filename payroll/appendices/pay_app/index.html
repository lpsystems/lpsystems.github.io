Data Dictionary

The following is a list of the main database tables within Payroll Master. The tables are set out in more detail in the remainder of this appendix.

Not all the database tables used by Payroll Master are listed here. Only those tables that you are likely to use in reports and formulae are included. Please contact L&P Systems Limited for further details if required.

Table Name Table Description
py_bacs BACS Details
py_cheque Cheque Details
py_elembat Pay Element Batch Body
py_element Pay Element Details
py_hist Member's Payroll History
py_histories Member's Historic Payroll Details
py_melem Member's Pay Elements
py_sch Payroll Scheme Details
py_ytd Member's Year To Date Details

TABLE NAME : py_bacs

DESCRIPTION : BACS Details

Field Name Description Type
pyb_eftsub Submission Number INTEGER
pyb_scheme Scheme Code CHAR(4)
pyb_member Member Number INTEGER
pyb_mem_name Member Name CHAR(50)
pyb_net_pay Net Pay DECIMAL(10,2)

TABLE NAME : py_cheque

DESCRIPTION : Cheque Details

Field Name Description Type
pyc_eftsub Submission Number INTEGER

pyc_mem_name Member Name CHAR(50)
pyc_net_pay Net Pay DECIMAL(10,2)

TABLE NAME : py_elembat

Field Name Description Type

pel_order Order Number SMALLINT

DESCRIPTION : Pay Element Details
