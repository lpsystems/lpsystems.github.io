Introduction

This program allows maintenance of the Scheme file. This is where you will find data relating to a scheme. There are two screens in Scheme File, which can be accessed via the ‘Menu’ menu option, described in more detail later in this section. You are not able to Add new schemes in this program. This can be done through the Scheme File in Pension Master.

The different screens of information that you may find in the Scheme File are as follows:

  • Scheme’s Details
  • Scheme’s Bank Details

The following options and screen appear when this program is selected.

  • Query
  • Update
  • Menu
  • Report
  • Exit

Scheme Details Screen

Query/Search For A Scheme

Click ‘Query’. The following pop-up window is displayed:

Query Scheme Details

You may select records using the Scheme Code field. You may also use any combination (or even partial combination) of these fields to select the required scheme record(s). Click the lookup icon for a list of scheme codes.

Once you have entered your selection criteria, click ‘OK’ to proceed and retrieve the records from the database or click ‘Cancel’ to abort the selection and return to the main menu.

If a match with the selection criteria is found then the first record will be displayed to the screen and a message displayed at the bottom of the screen:

Record 00001 of 0000x

If you wish to view any other screen in Scheme File, click ‘Menu’, then select the appropriate screen by moving the highlight bar and click ‘OK’.

Viewing Scheme Details

To view the other records selected click ‘Next’ to view the next record, and ‘Previous’ to view the previous record. If ‘Next’ is chosen and there are no records after the current record an error message is displayed. If ‘Previous’ is chosen and there are no previous records an error message is displayed.

Updating A Scheme

Scheme’s Details

Click ‘Menu’, then double click “Scheme’s Details screen” and then click ‘Update’.

The Scheme’s Details screen is used to hold the address details and tax office details of the scheme.

Enter the fields required as follows:

ADDRESS LINES & POSTCODE - Enter the scheme’s address and postcode (if known). These will be used on the P14 and P60 forms in the Year End process.

TELEPHONE & FAX NUMBER - Enter the telephone and fax number of the scheme (if known).

E-MAIL ADDRESS - Enter the scheme’s e-mail address (if known). This must contain the ‘@’ symbol, which is present in all e-mail addresses.

TAX NAME - Enter the name of the Tax Office used for this scheme. This will be used on the P14 and P60 forms in the Year End process.

TAX DISTRICT & REFERENCE - Enter the district and reference code for the tax office used for this scheme.

PAYSLIP MESSAGE? - Tick this field if you wish to be prompted during the payroll run to enter a message on each member’s payslip.

PAYSLIP TOLERANCE - Enter a tolerance for member’s payslips. If a member’s Net Pay is in excess of the tolerance when compared to the member’s previous Net Pay, then a payslip will be printed in the payroll run. If the payroll run is the first run of the new tax year then payslips will be printed regardless of what the tolerance is. This field will default to 0.00.

PAY IN ADVANCE? - Tick this field if this scheme pays their pensioners in advance. This will allow members whose date pension commenced is after the current payroll run date, but before the next payroll run date, to be included in the current payroll run.

Click ‘OK’ to save the Scheme’s Details, or click ‘Cancel’ to exit the screen without saving the details entered.

Scheme’s Bank Details

Click ‘Menu’ then double click “Scheme’s Bank Details”. Then click ‘Update’.

The following screen appears when selected.

Scheme Bank Details Screen

The Scheme’s Bank Details screen is used to hold the bank details of the scheme.

Enter the fields required as follows:

ACCOUNT TYPE - Select the Account Type.

BANK NAME - Enter the name of the scheme’s Bank/Building Society.

BANK ADDRESS LINES & POSTCODE - Enter the address and postcode of the scheme’s Bank/Building Society.

ACCOUNT NAME - Enter the name of the account of the scheme.

ACCOUNT NUMBER - Enter the account number of the scheme.

SORT CODE - Enter the sort code of the scheme’s account. The “-“ seperators are added automatically.

ROLL NUMBER - Enter the Building Society Roll Number. This field will only be displayed if the Account Type, that has been selected is Building Society (BSOC).

STATEMENT MESSAGE - Enter an appropriate message which will be displayed on the bank statement of those members who are to be paid by BACS. If a message is not entered, then the message will default to the first 18 characters of the Scheme Name.

Click ‘OK’ to save the Scheme’s Bank Details, or click ‘Cancel’ to exit the screen without saving the details entered.

Printing Scheme Details

Click the ‘Report’ icon.

The following window now appears:

Select Report Options Window

The options are as follows:

Print current screen Produce listing of scheme details only for the screen the user is in.
Print full scheme details Produce listing of all scheme details, including all screens.

You can also select the records to print in this dialog box. Further details of these options can be found here.

Select the appropriate option and click ‘OK’. Click ‘Cancel’ to exit the printing or ‘Help’ to display this help text.

Full Scheme Details

If you chose to print a Full Scheme Details Report, a window will now appear which asks which screens you wish to print for this scheme. All screens are defaulted to Yes.

If you wish to alter these defaults for this printout, simply tick the box against the desired screen for yes or leave the box blank for no.

Click ‘None’ to de-select all screens for the printout.

Click ‘All’ to select all screens for the printout.

When you have finished, click ‘OK’ to proceed to the printing stage. The screen selection screen is as follows:

Report Selection Screen

The options for printing are described in the document Printing From Payroll Master.

Instigating A Workflow Action

Select the ‘Workflow’ button to instigate a workflow action for the current scheme. A scheme record must be selected using the Query option before you can instigate a workflow action.

The following pop-up window is now displayed:

Workflow Instigation Popup

Enter the workflow code for the workflow action you wish to instigate or click the lookup icon for a list of the available workflow codes. To set up a workflow, use the Workflow Management option.

You can click ‘OK’ to continue or click ‘Cancel’ to abort the this workflow action.

Navigating Between Screens

Click ‘Menu’. A pop-up menu will appear on the screen which allows the user access to the two screens of the Scheme File. You simply highlight the desired screen and click ‘OK’. You can also click ‘Cancel’ to abort the screen selection process.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
