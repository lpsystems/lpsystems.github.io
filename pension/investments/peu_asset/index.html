Introduction

This program allows you to transfer the existing units for a member to the scheme suspense account. It will transfer all or a % of units for a member to the scheme suspense account and write a negative unit entry to each of the members funds so that the members total unit holdings.

It is envisaged that you might use this program, for example, to transfer any remaining units for a member to scheme level after the member has taken a refund of the value represented by units bought with their employee ordinary contributions.

The following options and screen appear when this program is selected.

  • Run
  • Undo
  • Exit

Asset Transfer Selection Screen

Run

Click ‘Run’.

SELECTION CRITERIA - Select the group of members you require to run the Asset Transfer for using the Scheme, Company, Category, Status, Special Code and Member fields. When the selection criteria have been entered click on ‘OK’ to proceed or click on ‘Cancel’ to return to the main menu.

The following screen is displayed:

Asset Transfer Details Screen

Enter the details as follows:

EFFECTIVE DATE - Enter the date due for your asset transfer. Only units bought up to an including this date will be included in the asset transfer.

COMMENTS - Enter any comments you may have about this asset transfer. These will then be saved to the scheme suspense account against each suspense account entry generated.

Now select the contribution types required, tick ‘all’ to select them all. The default is 100% but you can changed for each contribution type.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The asset transfer process will now run.

Note that only contribution types that are marked as non Protected Rights contribution types will be included in the asset transfer.

If the asset transfer is successful, a report will be generated which shows details of the units transferred to the scheme suspense account, broken down member by member.

The program will then return to the main menu.

Undo

This is essentially the reverse of the asset transfer run i.e. it reverses an asset transfer for a group of members.

Click on ‘Undo’.

The same two screens as for the ‘Run’ option are shown except that the comments field is not required.

All suspense account entries for the selected members and for the entered effective date are removed from the system.

The program will then return to the main menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
