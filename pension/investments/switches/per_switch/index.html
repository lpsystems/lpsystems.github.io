Introduction

This module allows you to produce a list of any switches that have been instigated for the selected group of members, where the money has not yet been dis-invested from the old investment fund.

This will allow you to keep track of which switches are still outstanding at any point of time.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

Enter the selection criteria required (see Selection Criteria). When the selection criteria has been entered click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the proceed menu and abort report selection.

When the report has been processed and records are found to match the selection criteria the report is displayed. You can now view and/or print the report as required.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
