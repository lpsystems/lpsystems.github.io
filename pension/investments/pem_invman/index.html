Introduction

In this program you can hold further data on the investment manager that can then be used in letters or reports. An investment manager code must already have been entered using Codes

  • Query
  • Update
  • Exit

Updating Investment Managers

Before Updating, a record must be selected through the Query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The Manager Code cannot be changed as this is the key to the Investment Manager file. When the changes have been made to the record click ‘OK’ to save the changes to file, or click ‘Cancel’ to exit the update option without saving the changes.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
