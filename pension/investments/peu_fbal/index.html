Introduction

This module allows you to report on members who have had a lifestyle event since the last date they invested contributions. It shows you the current allocation of the members investments, and the proposed new split of investments after re-investing according to the appropriate lifestyle matrix.

A lifestyle event is where a member crosses a boundary within a lifestyle matrix e.g. if the lifestyle matrix sets out that at age 50 they must switch investments into cash based funds, when the member reaches age 50, they have a lifestyle event.

The following options and screen appear when this program is selected:

  • Run
  • Exit

Fund Balancing Selection Screen

Run

Click ‘Run’.

This screen allows you to select the member(s) to be included in the fund balance report:

SELECTION CRITERIA - Select the group of members you require to run using the Scheme, Company, Category, Status, Special Code and Member fields. When the selection criteria have been entered click on ‘OK’ to proceed or click on ‘Cancel’ to return to the main menu.

The following screen is displayed:

Run Date Screen

Enter the details as follows:

RUN DATE - Enter the run date for your report.

When you have entered these details, click on ‘OK’ to proceed or ‘Cancel’ to return to the main menu.

The following window is now displayed:

For each fund that the selected members (who have a lifestyle event) have investments in, this allows you to input a unit sell price that will be used to calculate the value of the units to sell for each member, and the resultant split of re-invested contributions.

Enter unit prices for all applicable funds here, then click on ‘OK’. Click on ‘Cancel’ to return to the main menu without doing any further processing.

A report showing all those members who have had a lifestyle event since the date they last made an investment on their Investment History screen is now produced. You should use this to action the appropriate switches for the members concerned using Investment Switches.

The program now returns to the main menu.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
