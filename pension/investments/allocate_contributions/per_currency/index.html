Introduction

This module allows you to produce a report on contributions allocated. It is the same report as the alloaction report but the data is sorted by contribution type and what currency that they are going to be converted into and that is then totalled.

The following options and screen appear when this program is selected.

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

Enter the selection criteria required (see Selection Criteria). When the selection criteria has been entered click ‘OK’ to proceed. You can click ‘Cancel’ to return to the proceed menu and abort report selection.

You will now be asked the date due, date received and the transaction type.

</div>

When the report has been processed and records are found to match the selection criteria the report is displayed. You can now view and/or print the report as required.

The options for printing are described in the document Printing.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
