Introduction

This module allows you to produce a list of any contributions that have not yet been allocated to investment funds for a selected group of members.

The following options and screen appear when this program is selected.

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

Enter the selection criteria required (see Selection Criteria). When the selection criteria has been entered click ‘OK’ to proceed. You can click ‘Cancel’ to return to the proceed menu and abort report selection.

You will now be asked if you want a Summary or Detailed report. These are as follows:

Summary This produces a report with one line per distinct date due and a count of the number of unallocated contribution entries for the selected members for that date due.
Detailed This is the same as the summary report above except that full details of all unallocated contributions, member by member are printed.

When the report has been processed and records are found to match the selection criteria the report is displayed. You can now view and/or print the report as required.

The options for printing are described in the document Printing.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
