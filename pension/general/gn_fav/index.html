Introduction

This document describes the ‘Favourites’ menu tree which is available to all users of Pension Master.

This feature allows the user to access and setup menu options which they most commonly use. Clicking the ‘Favourites’ disclosure triangle displays a list of menu options:

Setting Up the Menu Options

Click ‘Favourites’ and the following window will be displayed:

Double click an item on the Main Menu list to include it in your list of Favourites.

To remove an item from your Favourites double click it.

Click ‘OK’ to save the details.

Click ‘Cancel’ to close the window without saving any new details.

Click ‘Help’ to display this help text.
