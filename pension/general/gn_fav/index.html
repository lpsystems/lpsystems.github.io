Introduction

This document describes the ‘Favourites’ button which is available to all users of Pension Master.

This feature allows the user to access and setup menu options which they most commonly use. Clicking ‘Favourites’ displays the following window.

Setting Up the Menu Options

Click ‘Update’. The following window will be displayed:

Use the ‘Add Row’ and ‘Delete Row’ buttons, shown to the top right of the screen, to insert or remove lines from the list. If the user wishes to add new menu options then they can click ‘Lookup’ to get a valid list of menu options for that user. Click ‘OK’ to save the details.

Click ‘Cancel’ to close the window without saving any new details.
