Search Lists

This document describes the featues common to all lookup or search lists in Pension Master.

To call up a search list, click on the lookup icon. This will open up a window similar to that shown below:

If there are more values in the search list that fit in the window, a scrollbar will appear to the right of the list. Use this scrollbar or the arrow keys to navigate around the list.

If there is more than one column of data displayed in the window, then you can click on the title of a column, and the data will be ordered by that column.

Use the mouse or the arrow keys to select the option you require, then click ‘OK’.

Click ‘Cancel’ to close the search list without selecting an item.

Click ‘Help’ to display this help text.
