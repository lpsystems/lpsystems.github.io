Introduction

This option is presented to you every time you select a print option from Pension Master.

Selecting Records To Print

Some print options (specifically those in one of the main programs in Pension Master e.g. Members) will prompt you to enter which records you wish to print with a window as follows:

The options are:

Print current record Print only the record that is currently displayed to the screen.
Print all selected records Print all records as selected by the Query menu option on the program’s main menu. For example, in Members, select the Query option and choose all members in scheme 0000. This print option will print details for all the members selected.
Select records to print Select another group of records to print.

Select the desired option, then click OK to proceed, Cancel to close the window and cancel the print request or Help to see this help text.

The Report Viewer

Your report is automatically displayed to the screen via the Pension Master Report Viewer:

Report Viewer

The report viewer displays one page of your report at a time. The current page and last page are displayed at the top of the screen e.g. ‘Page 1 of 20’.

To navigate between the different pages, use the ‘First Record’, ‘Previous Record’, ‘Next Record’ and ‘Last Record’ toolbar icons.

If you wish to jump straight to another page of your report, click the ‘Search’ icon on the toolbar. A pop-up window will be displayed asking you for the page number you wish to display. Enter a valid page number and click ‘OK’ to display that page. Click ‘Cancel’ to abort the jumping to a page function.

Click the ‘Print’ icon on the toolbar to print your report. You can print the report to a file or to a printer.

Click ‘Help’ to display this help text.

Click ‘Exit’ to exit the report viewer.

Print Options Dialog

This window is opened whenever you click ‘Print’ and allows you to:

  • select whether you wish to save the report to a file or print it to a printer.
  • change default printers.
  • print this report to a printer other than your current default printer.
  • select which portions of the report you wish to print.

Printing To A File

If you choose to print to a file, click ‘Select File’ to open the following window:

Enter the filename you wish to print to and then click OK. Click Cancel if you do not wish to specify a filename at this time.

You will then be returned to the Print Options Dialog Box described above.

The file that is to be printed to cannot contain any characters such as “/” or “". The file will be created in the c:\ root directory on your PC’s hard disk.

The report will then be saved to the top of the C:\ drive on your PC with the filename you specified e.g. if you enter member.txt as the filename, the report will be saved as c:\member.txt.

Printing

To An Alternative Printer

If you wish to print to a printer other than the printer you normally print to, or wish to set your default printer to be another printer, click ‘Select Printer’ to open the following window:

Select the printer that you wish to print to and click OK (to the right of your Pension Master screen). Your currently selected Pension Master printer is shown under ‘Current Printer’.

If you wish to set your default printer to another printer, click on the desired printer, then click ‘Default’. Your default printer will be changed for all future reports that you print and your current report will be sent to your new default printer.

Print Range

You can either print the whole report, just the currently displayed page or a range of pages. To do this enter the required option in the ‘Print Range’ section of the screen.

If you choose to print a range of pages, enter the start and end page numbers in the ‘From:’ and ‘To:’ fields.

Specifing Where To Create

Report Files

You can control where your report files are created by using the LP_PRINT_PATH environment variable which can be set in your Pension Master startup batch file (when running on Windows NT) or in your .profile, or /etc/profile (when running on Unix or Linux). e.g. if you wish to create all reports in c:\temp, put the following line in your Pension Master startup file (on NT):

set LP_PRINT_PATH=c:\temp

If LP_PRINT_PATH is not set, Pension Master will attempt to create your report according to the following rules:

Windows NT Unix or Linux
in path set by %TEMP% in path set by $DBTEMP
in path set by %TMP% in /tmp

If Pension Master cannot print to any of these locations, an error message will be displayed to the user informing them of this. This error message suggests the problem is due to a permissions error and asks the administrator to contact their Systems Administrator.
