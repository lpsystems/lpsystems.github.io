Introduction

You will need to logon to Pension Master with a valid Pension Master username and password.

Logging on to Pension Master

The following screen allows you to log onto Pension Master using a valid Pension Master username and password.

Logon

Enter the fields as follows:

DATABASE - Select the name of the Pension Master database that you wish to connect to or select the Default Pension Master Database.

USER NAME - Enter your username as assigned to you by your System Administrator.

PASSWORD - Enter your password.

Once sufficient log in details have been entered, you will be granted access to Pension Master.

If there are any diary items for you, the Diary File Maintenance screen will now be shown. Once you have finished with the diary file, click ‘Exit’.

Pension Master Main Menu

The Pension Master main menu is now displayed:

Main Menu

The options you see on this menu will depend on those set up for you by your System Administrator but would typically be those shown in the screenshot above.

To expand a branch of the menu structure click on the disclosure triangle to the left of the menu option.

Menu options are indicated by a dot to the left of the option discription. Select a menu option by double-clicking on the menu option text.

Menu

The quick menu enables you to enter the text of a menu and they by clicking on the option it brings you straight there witout having to go through the menu tree. To access it you type ‘M’ in the main screen and the following is displayed. Simply type what you require and then select.
