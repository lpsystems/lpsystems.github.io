Release Notes 4.5.4

New Features

  1. Fixed a bug in Diary File Maintenance where if there was only one item in the diary and the user tried to delete this, they were not allowed to do so as the system was saying Value Must Be Entered.
  2. Added support for logging out of all Novell Netware servers at the end of the pension.sh startup script for those users connecting to a Novell Netware network whilst using Pension Master.
  3. Fixed a bug in Category File Maintenance that ocurred when you update some category details then switch to another screen in Category File Maintenance without using the Query option first. The details on the screen that was switched to were not being displayed properly due to the recent incorporation of the Entry/Exit Criteria screen in Category File Maintenance.
  4. Added F8 lookup facility to the Query screen in Member File Maintenance on the following fields:
    • Scheme Code
    • Company Code
    • Category Code
    • Status Code

    Also removed the Company Name field from the Query screen as the F8 lookup on the Company Code field replaces this.
