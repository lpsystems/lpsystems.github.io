Release Notes 5.0.9

Release Notes

Search Lists

All scheme, company, category and status pop-up picklists are now sorted in alphabetical order on the appropriate description field e.g. schemes are sorted by scheme name, companies are sorted first by scheme code and then by company name.

Formula File Maintenance

Added various new functions to allow for adding and subtracting years, months and days from a date within yourt formulae:

addyear(date,num)/b>

OVERVIEW

Use the addyear function to add a fixed number of years onto a given date.

SYNTAX

addyear(date,num)

EXPLANATION

date     date to which the years are to be added

num     number of years to be added to date

NOTES

  1. date must be a field reference to the member file (pe_mem), a constant date entered in the format YYYYMMDD, where YYYY is the year, MM is the onth and DD is the day or a reference to the result of a previous formula line (as long as it evaluates to a valid date) in the format @line_num e.g. @2.
  2. num must be a whole number of years between 0 and 500.

EXAMPLES

addyear(pm_dob,18)
addyear(19950122,1)
addyear(@1,20)
