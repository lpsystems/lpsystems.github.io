DC Generic Benefit Statement

The DC Generic Benefit Statement has been set up via the normal quotation system in Pension Master. The quotation type is called, DC Generic Benefit Statement. Users do not have to set up the quotation type as it will be automatically added to all categories of non Defined Benefit (DB) type schemes. This quotation type cannot be updated as the specific calculations for the statement are automatically generated. A configuration screen, The DC Generic Configuration Screen, holds information in relation to the calculation of benefits. The results of the calculations can then be used in Mail Merge Reports and merged into your own benefit statement template. If the DC Generic Quotation does not have all the calculations you require, you can set up a trigger quote with the additional quotes that will trigger the DC Generic quote to run before it.
