Introduction

This program allows the user to load the opt out flag onto Pension Master for the members in the data file. The data file must contain data in the format NI_Number,Opt_Out_Flag. Only members that already exist on Pension Master will have their record updated. For any members found in the data file, who don’t exist on Pension Master and who have chosen to take part in the Combined Pension Forecast process, must be added onto Pension Master as a new member prior to loading on this data.

The following options and appear when this program is selected:

  • Load
  • Exit

Opt Out Load Screen

Run A Load

Press L, or select the ‘Load’ menu option.

SCHEME CODE - Enter the scheme code for the scheme the members in the data file belong to. You can click on the lookup icon to see a list of all valid scheme codes.

FILENAME - Enter the filename for the data file you wish to load the data from. You can click on the lookup icon to search for the file.

DELIMITER - Enter the delimiter used in the data file. You can click on the lookup icon to see a list of delimiters.

A message is displayed at the bottom of the screen:

Checking data

The program is checking that the data is all present and correct. For example, that the NI Number is present and that the opt out flag takes the value of Y or N. If any of the data isn’t correct then an exceptions report will be produced and the Opt Out Load is abandoned.

If all the data is correct, then a report is produced listing out all the information from the data file.

The options for printing are described in the document Printing from Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
