Introduction

This program produces a data file for members who fall in the selection criteria entered by the user and whose ‘Opt Out’ field, on the Combined Pension Forecast screen in Members, is unticked. The data file contains data such as member’s NI Number, surname, forename, date of birth and sex. The data file will then be sent to the Department of Widows and Pensions (DWP) as part of the initial stage of the CPF process.

The following options and appear when this program is selected:

  • Proceed
  • Exit

CPF Export Screen

Run An Export

Press P, or selected the ‘Proceed’ menu option.

You will be prompted to choose between a Provisional or Actual Export. Both options work exactly the same, however, the actual will create the datafile and save the file number created.

Enter the selection criteria required (see Selection Criteria). When the selection criteria has been entered click on ‘OK’. You can click on ‘Cancel’ to return to the proceed menu and abort report selection.

The media type must be selected. You can click on ‘lookup’ to give you a list of valid media types.

If there are no records to match the selection criteria, the following message is displayed to the screen:

There are no records to match the selection criteria

If there are records, then a message is displayed at the bottom of the screen:

Checking member’s data

The program is checking that the data is all present and correct. For example, that the NI Number is in the correct format of AA123456A, that the first character of the surname is alphabetic etc. If any of the data isn’t correct then an exceptions report will be produced and the CPF Export run is abandoned.

If all the data is correct, then a message will be displayed at the bottom of the screen:

Processing report … Please Wait

A report and a data file will now be produced, and the report will be displayed to the screen. This report will list the member’s who have been included in the data file.

The options for printing are described in the document Printing from Pension Master.

The name of the data file will be displayed to the screen. This data file can be located in the programs driectory on the Pension Master server.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
