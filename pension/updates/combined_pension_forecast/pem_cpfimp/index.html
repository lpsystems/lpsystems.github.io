Introduction

This program allows Combined Pension Forecast (CPF) data to be imported from a data file that has been sent from the Department of Widows and Pensions (DWP). The data file will contain information such as the members NI Number, Surname, Forename, Date of Birth, sex, and all Combined Pension Forecast data such as current basic state pension, projected basic state pension etc.

The following options and appear when this program is selected:

  • Proceed
  • Exit

CPF Import Screen

Run An Import

Press P, or select the ‘Proceed’ menu option.

Enter the name of the bulk reply file. You can click on the lookup icon to open up the following window for you to select the file you need.

A message is displayed at the bottom of the screen:

Importing data

The program is checking that the data is all present and correct. For example, that the NI Number and sequence number found in the reply file matches those in the request file from the CPF Export program. If any of the data isn’t correct then an exceptions report will be produced and the CPF Import run is abandoned.

If all the data is correct, then a report is produced listing out all the information from the data file.

The options for printing are described in the document Printing from Pension Master.

Each line of data returns a forecast status of 00 to 08. 00 is the only forecast status that indicates that the members data has been successfully processed this time. Therefore, only members with forecast of 00 will be imported from the data file into the Combined Pension Forecast screen in Members.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
