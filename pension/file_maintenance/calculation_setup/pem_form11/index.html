Formulae

#pay_elements_field(code)

This is only available to Payroll Master users.

OVERVIEW

Use this to refer to the value of a field on the Member’s Pay Elements screen for the specified code. This will be taken from the pay elements file (py_melem).

SYNTAX

#pay_elements_field(code)

EXPLANATION

  • code - refers to the Elements Code that you wish to retrieve a Pay Elements Field for. This must be in UPPERCASE and must exist as a valid Pay Element Code. If a member does not have a Pay Elements record with a matching code, this function call will return a value of 0.

NOTES

  1. pay_elements_field must be a valid field on the Pay Elements file (py_melem).

EXAMPLES

#pym_amount(PA766)
#pym_ytd(PD12)

paymem_hist(hist_fld,date)

This is only available to Payroll Master users.

OVERVIEW

Use this function to extract member’s pay history details depending on the history field and date entered.

SYNTAX

paymem_hist(hist_fld,date)

EXPLANATION

  • hist_fld - is a reference to a pay history field.
  • date - refers to the Date Processed on the pay history table.

NOTES

  1. The pay history field must exist on the Member’s History table (py_hist).
  2. The date entered must either be in the format yyyymmdd or be a reference to a previous line.

EXAMPLES

paymem_hist(pyh_gross,20020101)
paymem_hist(pyh_net,@1)

#payroll_history_field(date)

This is only available to Payroll Master users.

OVERVIEW

Use this to refer to the value of a field on the Member’s History screen for the specified code. This will be taken from the history file (py_hist).

SYNTAX

#payroll_history_field(date)

EXPLANATION

  • date - refers to the Date Processed on the history table. If a member does not have a history record, this function call will return a value of 0.

NOTES

  1. The date entered must either be in the format yyyymmdd, a reference to a previous line or a valid date from the Basic Member Details screen.
  2. payroll_history_field must be a valid field on the history file (py_hist).

EXAMPLES

#pyh_gross(20020101)
#pyh_net(@1)
