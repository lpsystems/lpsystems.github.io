Formulae

int @line_number

OVERVIEW

Use the int function to round a value of a particular line in a formula down to the lower whole number.

SYNTAX

int @line_number

NOTES

  1. int will round the value of a formula line down to the lower whole number. e.g. int 2.123 = 2.
  2. the result of a formula will always be displayed to 2 decimal places by default. If you wish to have the result displayed as a whole number then use the format(field,Nn) function.

EXAMPLE

int @2
