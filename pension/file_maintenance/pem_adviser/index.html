Introduction

Only users set up as Supervisor status in Users may access this program.

The type of access granted can be view only or full access. For the view only option, you will not be permitted access to any menu option that allows you to add/update/delete data.

If you do not have Supervisor status, you will not be permitted access to this program.

This program allows maintenance of the Adviser file. This is used to:

  • Set up each adviser that you deal with. This allows you then assign the appropriate adviser to the scheme record in Schemes.
  • Record details of each advisers address, contact names and addresses which can be used to generate mail merge letters.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Update
  • Delete
  • Exit

Adviser Details Screen

Add An Adviser

Click ‘Add’.

ADVISER CODE - Enter a code for your new adviser record. The code entered must not already exist on the adviser file. If a valid code is not entered an error message will be displayed and adviser code will have to be re-entered.

ADVISER NAME - Enter the name of the new adviser. A value must be entered in this field and will be used in subsequent lookups for the adviser details.

ADDRESS - Enter the adviser address and postcode as appropriate.

TELEPHONE - Enter the telephone number of the adviser.

FAX - Enter the fax number of the adviser.

EMAIL - Enter the adviser’s E-Mail address.

When the adviser details have been entered click ‘OK’ to save the details to the adviser file. You can click ‘Cancel’ at any stage to exit the add option without saving the details.

Query/Search For An Adviser

Click ‘Query’. The following pop-up window is displayed:

Adviser Query Screen

You may select records by using the adviser code field. You may also use any combination (or even partial combination) of these fields in order to select the required company record(s).

If you are unsure of the code(s) to Query, click the lookup icon to see a list of Adviser codes.

Once you have entered your selection criteria, click ‘OK’ to proceed and retrieve the records from the database or click ‘Cancel’ to abort the selection and return to the main menu.

If any match to the selection criteria is found then the first record will be displayed to the screen and a message displayed to the bottom of the screen.

Record 1 of 2

Viewing Advisers

To view the other records selected press N to view the next record, and P to view the previous record. If N is chosen and there are no records after the current record an error message is displayed. If P is chosen and there are no previous records an error message is displayed.

Updating Advisers

Before Updating, a record must be selected through the Query option, and an error message is displayed if a record for update has not been selected.

Click ‘Update’.

The adviser code cannot be changed as this is the key to the adviser file. When the changes have been made to the record click ‘OK’ to save the changes to file, or click ‘Cancel’ to exit the update option without saving the changes.

Deleting Adviser Details

Before Deleting, a record must be selected through the Query option, and an error message is displayed if a record for deletion has not been selected.

Also, the adviser to be deleted must not be used by any scheme on your Pension Master system. If it is, a warning message will be displayed to the screen informing you of this and you will not be able to delete the adviser code.

Click ‘Delete’.

You will be asked to confirm and then re-confirm deletion of this Adviser.

Printing Adviser Details

Click the ‘Report’ icon.

If records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen

If no records have been selected in the program before the report option was selected the following window is displayed.

Print Options Screen

Record Prints the current screen Record.
Query Prints the all the selected records.
Select Allows selection of records for the report using Adviser Code field.
Exit Closes the window and leaves the report option.

If the Select option was chosen and no records are found to match the selection criteria entered, an error message is displayed and new selection criteria should be entered.

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
