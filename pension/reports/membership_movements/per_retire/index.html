Introduction

This program produces a report of active and deferred members who are due to retire between two dates entered by the user. This can be used to list members who have retired over the past few years, or to list impending retirements in the future.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Member Selection Screen

Run A Report

Click ‘Proceed’.

Enter the selection criteria required to select the members you wish to output to the report (see Selection Criteria). When the selection criteria has been entered click on ‘OK’ to proceed to the second screen. You can click on ‘Cancel’ to return to the proceed menu and abort report selection.

When the selection criteria have been entered a second screen is drawn as follows:

Date Selection Screen

REPORT DATE - The report selection dates default to producing the report for the year immediately prior to todays date, but can be overwritten by any dates in the format dd/mm/yyyy or ddmmyyyy.

You can click on ‘Cancel’ to exit input of report dates and return to the report menu.

The date field checked for the report is pm_npd - Normal Pension Date.

i.e. If Normal Pension Date >= Report From date and Normal Pension Date <= Report To date the member will be included in the report.

e.g. If a listing of members who are due to retire on or between 01/01/92 and 02/02/92 is required then enter 010192 in report from date and 020292 in report to date.

When two dates have been entered a message is displayed at the top of the screen

Processing report … Please Wait

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
