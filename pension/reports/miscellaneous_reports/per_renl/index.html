Introduction

This program allows the user to analyse the current state of play with renewals for a single broker, a group of brokers, or across your whole Pension Master system.

The report shows which schemes (and constituent companies within a scheme) have renewal data outstanding, which have renewals completed and also prints an renewal data overdue analysis so that you can see at a glance which schemes require following up in order to ensure that you process renewals in a timely fashion. The report is constructed by analysing the Renewal Details screen in Schemes.

For details of how to set up your broker codes, see Brokers.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

Enter the selection criteria required (see Selection Criteria). When the selection criteria has been entered click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the proceed menu and abort report selection.

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
