Introduction

This program allows the user to produce a listing of scheme renewals due, sorted by renewal date. Optionally, you can tell the report only to print the renewals due in a given month or range of months. We anticipate that you will find this report useful in planning out your work on impending renewals.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

Enter the renewal month required or leave this blank to select renewals for any month.

Click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the proceed menu and abort report selection.

Once the report has been produced, options are displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
