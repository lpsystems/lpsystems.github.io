Introduction

This program allows the user to produce a P45 for a selection of members. P45 information is extracted from the member’s records and used in the P45, along with any data entered into the 2nd screen.

The following options and screen appear when this program is selected:

  • Run
  • Exit

Selection Screen

Run A Report

Click ‘Run’.

SELECTION CRITERIA - Select the group of members you require to produce this report for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed. The scheme code must be entered.

You can click ‘Cancel’ to return to the main menu and abort the P45 report.

The following screen is displayed:

P45 Details Screen

FUND - Enter the investment fund for the investments you are reporting on. If you are unsure of what code to use, click the lookup icon to see a list of investment funds. If you want to report on all investment funds then enter ‘ALL’.

TRANSACTION TYPE - Enter the transaction type for the investments you are reporting on. If you are unsure of what code to use, click the lookup icon to see a list of transaction types. If you want to report on all transaction types then enter ‘ALL’

START DATE - Enter the start date for the range of investments you are reporting on.

END DATE - Enter the end date for the investments you are reporting on.

ORDER BY - Enter the order in which you wish the investments to be ordered by.

REPORT TYPE - Enter whether you wish the investments to be reported in a summary or detailed format.

When the details in this screen have been entered, click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the main menu and abort the investment reconciliation report. The report will now be created and reported out to Excel.

During the creation of the report, unit prices for the investment fund are retrieved for the start and end dates to allow for fund values to be calculated. If they are not known then the unit price effective at the date will be used instead.

The unit price for the end date will always be the unit price found on the system effective at the end date.

Once the report has been produced, a message will appear to the screen stating that the report has completed successfully.

SCHEME CODE - Enter the scheme code required. To get a list of valid scheme codes click on the lookup icon. When a valid scheme code has been entered the scheme description will be displayed.

LAST RUN DATE - When the scheme code has been entered, the last run date for that scheme is displayed in this field. You cannot enter into this field.

DUE DATE - Enter the date you wish to list member’s payroll due at, in the format dd/mm/yyyy or ddmmyyyy.

Click ‘OK’ to proceed. You can click ‘Cancel’ to return to the proceed menu and abort report selection.

When the selection is complete a message is displayed at the bottom of the screen:

Processing report … Please Wait

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

Click ‘OK’ to return to the report menu.

If records are found to match the selection criteria the report continues and is displayed to the screen.

The options for printing are described in the document Printing From Payroll Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
