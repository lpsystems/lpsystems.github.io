Introduction

This program allows the user to produce a listing of current preserved members for a selected group of members. The report can be used as a standard report which can be used for all schemes. Any extra columns that are required can be included by generating a similar report using the Member Report option under the Report Writer submenu.

The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

Enter the selection criteria required (see Selection Criteria). When the selection criteria has been entered click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the proceed menu and abort report selection.

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
