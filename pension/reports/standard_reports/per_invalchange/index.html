Introduction

This program produces a report on Investment Value Changes. It lists the units and the value of investments (including the unallocated amounts) at the start and end dates, that are inputted for the transaction types selected in the screen that appears in the report run. The contributions and units added during that period are shown along with the opening value and closing value of them. It then shows the earnings of the funds (ie end value - start value - contributions) over that period, for the member selection criteria input
