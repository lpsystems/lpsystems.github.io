Introduction

This program allows the user to produce a report that lists changes in all or a specific contribution type effective at a given date and the previous amount type on the contribution history screen where the difference is not equal to zero

  • Proceed
  • Exit

Selection Criteria Screen

Run A Report

SELECTION CRITERIA - Select the group of members you require to produce this report for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed.

You can click ‘Cancel’ to return to the main menu and abort the member amount change report.

The following screen is displayed:

Selection Screen

CONTRIBUTIONS - Click ‘All’ if you wish to report on all the contributions for the scheme, or click ‘Select’ to select which ones to report on.

TRANSACTION - Click ‘All’ if you wish to report on all the Transaction types for the scheme, or click ‘Select’ to select to report on.

DATE FOR COMPARSION - Enter the date to be used from the drop-down list.

EFFECTIVE DATE - Enter the effective date for the contributions you are reporting on.

When the details in this screen have been entered, click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the main menu and abort the member amount change report. The report will now be created and reported out to Excel.

Once the report has been produced, a message will appear to the screen stating that the report has completed successfully.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
