Introduction

This program produces a report out to Microsoft Excel of DC benefit details, as selected by the user, for a given group of members, for either all investment funds or just one selected fund, and for investments between a date range entered. The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Criteria Screen

Run A Report

SELECTION CRITERIA - Select the group of members you require to produce this report for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed.

You can click ‘Cancel’ to return to the main menu and abort the DC benefit certificate report.

The following screen is displayed:

Fund and Date Selection Screen

FUND - Enter the investment fund for the investments you are reporting on. If you are unsure of what code to use, click the lookup icon to see a list of investment funds. If you want to report on all investment funds then enter ‘ALL’. START DATE - Enter the start date for the range of investments you are reporting on.

END DATE - Enter the end date for the investments you are reporting on.

When the details in this screen have been entered, click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the main menu and abort the DC benefit certificate report. The report will now be created and reported out to Excel.

During the creation of the report, unit prices for the investment fund are retrieved for the start and end dates to allow for fund values to be calculated. If they are not known then the unit price effective at the date will be used instead.

The unit price for the end date will always be the unit price found on the system effective at the end date.

Once the report has been produced, a message will appear to the screen stating that the report has completed successfully.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
