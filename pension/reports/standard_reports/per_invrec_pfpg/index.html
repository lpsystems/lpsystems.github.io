Introduction

This program allows the user to produce a report on the funds, per member, date invested and the total units. This is relative to the date invested for each of the funds selected from the fund selection window, for the transaction types selected from the transaction type selection window, where date invested is greater than or equal to the start and less than or equal to the end date of the period indicated in the report run criteria, for the selection criteria input

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

You can enter ALL for all funds in the scheme or one unique fund or use the | operator for more than one funds ie ILAA1|ILAA2. Then enter your transaction type and the effective start and end dates. When the selection is complete a status bar is displayed in the screen

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
