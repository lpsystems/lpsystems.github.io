Introduction

This program produces a report out to Microsoft Excel for members, as selected by the user, where the latest entry in the Annual/Lifetime Allowance screen in Members either has the LTA % greater than the warning threshold entered by the user, or has the AI % greater than the warning threshold entered by the user, or has either or both of these percentages greater than the warning threshold entered by the user. The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Selection Criteria Screen

Run A Report

SELECTION CRITERIA - Select the group of members you require to produce this report for using the Scheme, Company, Category, Status, Special Code and Member Number fields (see Selection Criteria). When the selection criteria have been entered click ‘OK’ to proceed.

You can click ‘Cancel’ to return to the main menu and abort the lifetime allowance/annual increase warnings report.

The following screen is displayed:

Select Report Type Screen

REPORT TYPE - Select ‘Lifetime Allowance’ if you want to produce a report where the member’s latest entry in the Lifetime/Annual Allowance screen has got a LTA % which is greater than the warning threshold. Select ‘Annual Increase’ if you want to produce a report where the member’s latest entry in the Lifetime/Annual Allowance screen has got an AI % which is greater than the warning threshold. Select ‘Either/both of the above’ if you want to produce a report where the member’s latest entry in the Lifetime/Annual Allowance screen has got either a LTA % or an AI % which is greater than the warning threshold, or both of them are greater than the warning threshold.

WARNING THRESHOLD - Enter a warning threshold that will be used to compare the LTA % and/or the AI % in the Lifetime/Annual Allowance screen for each member in the selection criteria.

When the report type and warning threshold have been entered, click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the main menu and abort the lifetime allowance/annual increase warnings report. The report will now be created and reported out to Excel.

Once the report has been produced, a message will appear to the screen stating that the report has completed successfully.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
