Introduction

This module allows you to produce a list of contributions and units for the selected group of members between two user entered dates.

The following options and screen appear when this program is selected.

  • Proceed
  • Exit

Selection Screen

Run A Report

Click ‘Proceed’.

Enter the selection criteria required (see Selection Criteria). When the selection criteria has been entered click on ‘OK’ to proceed. You can click on ‘Cancel’ to return to the menu and abort report selection.

The following screen is displayed:

Details Screen

START DATE - Enter the date for your units to be reported from. This defaults to today’s date, less one year.

END DATE - Enter the date for your units to be reported to. This defaults to today’s date.

REPORT TYPE - Select Summary or Detailed as follows:

Summary This produces a report with one line per member and fund. The contribution and unit entries between the two user entered dates are totalled for each fund and member combination.
Detailed This is the same as the summary report above except that full details of all contributions and units, member by member are printed.

When the report has been processed and records are found to match the selection criteria the report is displayed. You can now view and/or print the report as required.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
