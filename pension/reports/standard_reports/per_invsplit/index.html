Introduction

This program allows the user to produce a report that lists the investment fund price, and the member’s investment fund value at the run date for every fund the member has a value in. It then reports on the latest ERORD only contribution type investment setup at the run date. It shows the fund, the effective date, the allocation rate and if a Lifestyle matrix is the investment choice is lists the matrix code.

Please noted that only the ERORD contribution type setup is reported on.

Selection Screen

Run A Report

Click ‘Proceed’.

This date is used as the effective date for the unit price, and the investment set up plus the date used for the unit value. When the selection is complete a status bar is displayed in the screen

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
