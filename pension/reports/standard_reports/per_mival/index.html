Introduction

This program allows the user to produce a report which provides basic member data and the member’s total investment value, for member’s that meet the member selection criteria input in the member selection criteria screen. The members total investment value on this report is the same as the total invested value that appears on the member’s investment values summary screen. It only includes those monies actually invested in units.

Selection Screen

Run A Report

Click ‘Proceed’.

When the report has been processed and no records are found to match the selection criteria an error message is displayed:

There are no records to match the selection criteria

If records are found to match the selection criteria the report continues and options are then displayed for printing.

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
