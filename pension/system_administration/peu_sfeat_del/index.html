Introduction

This program deletes a member’s special code in bulk relating to the standard selection criteria screen of scheme, company, category, status and group. For all members in the selection if they have the special code assigned to them they are produced out to a report before the user is asked if they wish to continue and delete the special codes from these members. The following options and screen appear when this program is selected:

  • Proceed
  • Exit

Remove Members’ Special Codes
