Introduction

Only users set up as a supervisor status in Users may access this program.

The type of access granted can be view only or full access. For the view only option, you will not be permitted access to any menu option that allows you to Add/Update/Delete data.

If you do not have supervisor status, you will not be permitted access to this program.

This program allows the System Administrator to set up and maintain additional databases within Pension Master.

If Pension Master has multiple databases, you can tell which one you are connected to by looking at the titlebar of the Pension Master window - it will have a (dbasename.dbs) at the end of it, where dbasename is the name of the database you are using.

The following options and screen appear when this program is selected.

  • Add
  • Update
  • Delete
  • Report
  • Browse
  • Exit

Databases Screen

Add A Database

Click ‘Add’.

DATABASE NAME - Enter a name for your database. This can be up to 10 characters long and cannot contain spaces or already exist as a database.

DATABASE DESCRIPTION - Enter a description for your database so that your users can easily identify the purpose of the database e.g. “Formulae Development Database”. This can be up to 50 characters long.

When the details have been entered click on ‘OK’ to save the message to file. You can click on ‘Cancel’ at any stage to exit the add option without creating a new database.

Update A Database

Click ‘Update’.

Enter or amend your database description and click on ‘OK’ to save the changes. You can click on ‘Cancel’ at this stage to return to the main menu without saving the changes.

Browse For A Database

Click on the ‘Browse’ button or the Browse icon on the toolbar to view details in any screen where the data does not fit on the screen.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate around the data.

Click on ‘OK’ or ‘Cancel’ to leave browse mode.

Delete A Database

Click ‘Delete’.

You cannot delete the default ‘pension’ database or the current database and an error message will be displayed warning you of this if you try to delete either of these databases.

Type in the name of the database you wish to delete and press Return.

Printing Database Details

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
