Introduction

Only users set up as system administrator status in Users may access this program.

If you do not have system administrator status, you will not be permitted access to this program.

This program allows users to set up workgroups which can then be used to sub-divide users into different administration teams or areas.

The following options and screen appear when this program is selected:

  • Add
  • Delete
  • Report
  • Browse
  • Exit

Workgroups Screen

Add A Workgroup

Click ‘Add’.

CODE - Enter a code for your workgroup. This can be up to 8 characeters long and will automatically be converted into uppercase by Pension Master. The workgroup code must not already exist and a warning message will be displayed if it is. The workgroup code also cannot be the same as any valid username as set up in Users.

DESCRIPTION - Enter a description for your workgroup e.g. “Pension Admin Team A”. This can be up to 50 characters long.

When the workgroup details have been entered click on ‘OK’ to save the message to file. You can click on ‘Cancel’ at any stage to exit the add option without saving the entered workgroup.

Browse For A Workgroup

Click on the ‘Browse’ button or the Browse icon on the toolbar to view details in any screen where the data does not fit on the screen.

Once in browse mode, use the arrow keys or the scrollbar on the right of the data to navigate around the data.

Click on ‘OK’ or ‘Cancel’ to leave browse mode.

Deleting Workgroup Details

Click ‘Delete’.

Under the Workgroup Code heading, enter the existing workgroup code that you wish to delete and click on ‘OK’ to continue.

If the workgroup code has already been assigned to a user of your system you will not be permitted to delete it. In this case, an error message will be displayed to the screen informing you of this and you will be returned to the main menu. Otherwise, a window will be displayed with the options ‘Yes’ and ‘No’.

If you do not wish to delete the record, click ‘No’. You will return to the program menu.

If you wish to delete the record, click ‘Yes’. A new window is displayed asking if you are sure.

If you do not wish to delete the record, click ‘No’. The program returns to the main menu.

If you wish to delete the record, click ‘Yes’. The record is deleted, and the program returns to the main menu.

Printing Workgroup Details

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
