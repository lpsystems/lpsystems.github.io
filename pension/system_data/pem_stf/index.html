Introduction

Only users set up as Supervisor status in Users may access this program.

The type of access granted can be view only or full access. For the view only option, you will not be permitted access to any menu optionn that allows you to add/update/delete data.

If you do not have Supervisor status, you will not be permitted access to this program.

This program allows users to record the main Annual Statutory Amounts:

  • Single Persons Basic State Pension
  • Married Persons Basic State Pension
  • Widowed Persons Basic State Pension
  • Disabled Persons Basic State Pension
  • Lower Earnings Limit
  • Upper Earnings Limit
  • Earnings Cap
  • Earnings Threshold
  • Employees Rebate %
  • Employers Rebate %

It also allows the user to record the Tax Simplification amounts to be held at system level:

  • Lifetime Allowance
  • Annual Allowance
  • Annual DB Multiplier
  • Pension Valuation Factor
  • Pension In Payment Multiplier
  • Change in TV Value
  • Refund Tax Allowance
  • DC Lifetime Allowance
  • MP Annual Allowance

It also allows the user to record contributions limits related to age.

These amounts are held against the appropriate year and, in the case of the Basic State Pension figures, the date the Basic State Pensions became effective from. These amounts can be selected in formulae using the appropriate syntax.

The following options and screen appear when this program is selected:

  • Add
  • Query
  • Update
  • Delete
  • Report
  • Exit

Annual Statutory Amounts Screen

Adding Annual Statutory Amounts

Click ‘Add’.

Use this option to add a new year’s Statutory Amounts.

EFFECTIVE DATE - Enter the effective date. It is only possible to amend this in Add mode.

SINGLE BSP - Enter the Single Persons Basic State Pension for the year selected.

MARRIED BSP - Enter the Married Persons Basic State Pension for the year selected.

WIDOWED BSP - Enter the Widowed Persons Basic Pension for the year selected.

DISABLED BSP - Enter the Disabled Persons Basic Pension for the year selected.

BSP EFFECTIVE DATE - Enter the date that the Basic State Pensions became effective from. The year of the date must match up with the year of the selected line.

LEL - Enter the Lower Earnings Limit for the year selected.

UEL - Enter the Upper Earnings Limit for the year selected.

EARNINGS CAP - Enter the Earnings Cap for the year selected.

EARNINGS THRESHOLD - Enter the earnings threshold for the year selected (UK only).

EMPLOYEES REBATE % - Enter the employees rebate % for the year selected (UK only).

EMPLOYERS REBATE % - Enter the employers rebate % for the year selected (UK only).

When the Annual Statutory Amounts have been entered click on ‘OK’ to save the figures to file. You can click on ‘Cancel’ at any stage to exit the update option without saving the entered Annual Statutory Amounts.

You will now be sent through the the second screen, where you can enter the Tax Simplification Amounts.

Tax Simplification Amounts Screen

LIFETIME ALLOWANCE - Enter the lifetime allowance for the year selected (UK only).

ANNUAL ALLOWANCE - Enter the annual allowance for the year selected (UK only).

ANNUAL DB MULTIPLIER - Enter the annual DC multiplier for the year selected (UK only).

PENSION VALUATION FACTOR - Enter the pension valuation factor for the year selected (UK only).

PENSION IN PAYMENT MULTIPLIER - Enter the pension in payment multiplier for the year selected (UK only).

CHANGE IN TV VALUE - Enter the change in TV value for the year selected (UK only).

REFUND TAX ALLOWANCE - Enter the refund tax allowance for the year selected (UK only).

DC LIFETIME ALLOWANCE - Enter the DC lifetime allowance for the year selected (UK only).

MP ANNUAL ALLOWANCE - Enter the MP annual allowance for the year selected (UK only).

Query Annual Statutory Amounts/Tax Simplification Amounts/Contribution Limits

You will now be sent through the the third screen, where you can enter the Contribution Limits.

Contribution Limits Screen

Click ‘Query’. The following pop-up window is displayed:

Annual Statutory Amounts/Tax Simplification Amounts/Contribution Limits Query Screen

You may select records by using the Year field.

Once you have entered your selection criteria, click ‘OK’ to proceed and retrieve the records from the database or click ‘Cancel’ to abort the selection and return to the main menu.

Update Annual Statutory Amounts/Tax Simplification Amounts/Contribution Limits

Click ‘Update’.

When the changes have been made to the record click ‘OK’ to save the changes to file, or click ‘Cancel’ to exit the update option without saving changes.

Delete Annual Statutory Amounts/Tax Simplification Amounts/Contribution Limits

Click ‘Delete’.

When the delete option is selected a window will appear with the options yes or no.

If you do not wish to delete the record click ‘No’. You will return to the program menu.

If you wish to delete the record, click ‘Yes’.

A new window is displayed asking if you are sure.

If you do not wish to delete the record click ‘No’. The program returns to the main menu.

If you wish to delete the record, click ‘ Yes’. The record is deleted and the program returns to the main menu.

Printing Annual Statutory Amounts/Tax Simplification Amounts/Contribution Limits

The options for printing are described in the document Printing From Pension Master.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
