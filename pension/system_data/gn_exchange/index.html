Introduction

Only users who belong to a User Group that has the correct permission can access this option.

If you do not have permission, you will not be permitted access to this option.

This program allows maintenance of the Exchange Rates that are used for multi currency schemes. In order to set up a currency for a fund you need to set up a fund manager code in the codes progam. The exchange rates are assigned to the fund and are used during the process of the members contrbutions. This is to allow for one or many rates across the system. All of the fund codes are displayed when you enter the screen first you need to scroll to the one you want to update before you press update.

The following options and screen appear when this program is selected:

  • Update
  • Checked
  • Exit

Exchange Rate Screen

Update Rates

Click ‘Update’.

Now simply enter the new rate for the currency. The available currency will be displayed in a drop-down list. Only rates that have been checked are used in the allocating and unit processing programs.

Check Rates

Click ‘Checked’.

To check a rate simply tick the box beside the rate. The same user that updated the rate can’t check the rate aswell.

Exit Program

Click ‘Exit’. This leaves the program and closes the current browser tab.
